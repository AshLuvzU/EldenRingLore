Welcome!

The goal of this project is to provide some insight intoon where lore can be discovered within the game of Elden Ring.
This will include places and items such as:

  • Divine Towers
  • Remembrances
  • Great Runes
  • NPCs (in there first location)
  • Locations important to story progression

Elden Ring is an open world RPG directed by Hidetaka Miyazaki with worldbuilding provided by the American fantasy writer George R. R. Martin. The game was released on February 25th, 2022; since then the game has recieved over 300 Game of the Year awards.
