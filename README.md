# EldenRingLore

The purpose of this map is to mark important locations within The Lands Between. 
Currently this will include NPCs important to the lore of the game (Rennala, Radhn, Morgott, and so on) and their dialog, locations such as divine towers, items such as remembrances and great runes, and other important things. 
There will also be pages dedicated to information on the different regions and NPCs, which should hopefully provide a bit more background than what I can provide in the popups.
